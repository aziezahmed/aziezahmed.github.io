Some personal programming projects

newsapi-cli

Python command line application for viewing news headlines from various sources. Built with news headlines from newsapi.org.

openweathermap-cli

A command line application for displaying weather details. Written in Python, built with weather data from openweathermap.org.

yahoofinance-cli

A command line application for displaying stock prices and other finance data. Written in Python, built with financial data from Yahoo!.

my-portfolio

A web application for monitoring the performance of your stocks and shares portfolio. Add details of your financial portfolio and see the daily performance. Built with AngularJS with financial data from Yahoo!

See "My Portfolio" demo!

watch-list

Another web application for monitoring the performance of your stocks and shares portfolio. This time it's built with AngularJS with financial data from markit on demand, Market Data APIs.

See "Watch List" demo!

flickr-gallery

A web application that takes a flickr feed and displays the thumbnails on the page. Made with Bootstrap, Backbone.js, Require.js and Underscore.js Templates

See "Flickr Gallery" demo!

kodi-remote

A web interface remote for osmc or kodi.

photo-gallery

A PHP page for turning a http web directory listing into a slideshow for your photos.
