Photo Gallery

I like to share my photos on my hard disk over HTTP on my local network. Rather than see the Apache directory listing, I wanted a simple way to view the photos sequentially. Place this single page php file in any web directory, and instead of seeing the directory listing you can go through the photos one by one.
